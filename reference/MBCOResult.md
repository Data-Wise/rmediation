# MBCO Result Class

A class representing the results of a Model-Based Constrained
Optimization (MBCO) test.

## Usage

``` r
MBCOResult(
  statistic = integer(0),
  df = integer(0),
  p_value = integer(0),
  type = character(0)
)
```

## Arguments

- statistic:

  Numeric test statistic value.

- df:

  Numeric degrees of freedom.

- p_value:

  Numeric p-value.

- type:

  Character string indicating the type of test.
